# Interactive Architecture Diagrams

Visualize complex architectures, flows, and state machines using Mermaid JS.

## Features

- Create system architecture diagrams
- Flowcharts and sequence diagrams
- State machines
- Real-time preview

## Development

This is a static single-page application. Simply open `index.html` in your browser or deploy to GitHub Pages.

## Deployment

GitHub Pages is configured via `.github/workflows/deploy.yml`. After enabling Pages in repository settings, the site will be available at:

```
https://personalbotai.github.io/interactive-architecture-diagrams/
```
